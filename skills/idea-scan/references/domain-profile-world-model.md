# Optional domain profile: World Models

Load this profile only when the research topic concerns learned world models, predictive representations, interactive video models, or model-based control.

## Scan families

Cover the relevant subset rather than treating “world model” as one method:

- latent dynamics and imagination-based control, including RSSM/Dreamer-style systems;
- value-equivalent or planning-oriented models, including MuZero/TD-MPC-style systems;
- predictive representation learning, including JEPA-style objectives;
- generative video and interactive simulation models;
- embodied, robotic, driving, and physical-world models;
- benchmark and evaluation work on persistence, controllability, and physical consistency.

## Problem axes

- compounding open-loop error and long-horizon consistency;
- action conditioning, controllability, and causal sensitivity to actions;
- persistent state, object permanence, and memory;
- stochastic or multimodal futures and uncertainty calibration;
- representation or posterior collapse;
- planning utility versus perceptual quality;
- model exploitation, reward errors, and termination prediction;
- data efficiency, transfer, scaling, and compute cost;
- mismatch between teacher-forced metrics and deployment behavior;
- evaluation gaps, benchmark leakage, and weak controls.

## Search expansion examples

Expand terms across mechanism and evaluation vocabulary:

- persistent memory -> state persistence, object permanence, recurrent state, retrieval, latent memory, long-context dynamics;
- action-conditioned -> controllable generation, action sensitivity, intervention response, policy-conditioned prediction;
- long horizon -> open-loop rollout, temporal consistency, compounding error, multi-step prediction;
- useful world model -> planning performance, control return, value equivalence, imagination accuracy, model exploitation.

Do not infer a research gap merely because two exact phrases have not appeared together.
