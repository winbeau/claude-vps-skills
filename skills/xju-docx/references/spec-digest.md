# 《新疆大学本科毕业论文（设计）规范及格式要求》速查摘要

> 提取自附件1原文（docx）。本文件是 xju-docx skill 的权威格式参照；排版实现见 `../scripts/xju_format.py`。
> 中文字号→磅：一号=26 二号=22 三号=16 小三=15 四号=14 小四=12 五号=10.5 小五=9 七号=5.5

## 页面

| 项目 | 要求 |
|---|---|
| 纸张 | A4，Word 排版 |
| 页边距 | 上 2.54cm、下 2.54cm、左 3.17cm、右 3.17cm |
| 页眉/页脚距边 | 页眉 1.5cm，页脚 1.75cm |
| 页眉 | 从封面起每页都有；小五号宋体居中；文字下划一条 0.5 磅横线，线长与页面齐宽 |
| 行距 | 全文无网格，1.5 倍行距（一级标题除外） |
| 打印 | 封面/声明/任务书/摘要/目录/致谢单面；正文/参考文献/附录双面 |

## 页码

- 封面：**不加页码**
- 声明、任务书、中英文摘要、目录：**罗马数字**（Ⅰ、Ⅱ、Ⅲ…）
- 正文、参考文献、致谢：**阿拉伯数字**，从正文第一页起连续编号至致谢结束
- 位置：页面底端居中，最简数字，无其他符号

## 标题与正文

| 层级 | 格式 |
|---|---|
| 一级标题 | **另起一页**，居中，三号宋体，单倍行距，段前空 3 行、段后空 2 行 |
| 二级标题 | 左对齐顶格，小三号宋体，1.5 倍行距，段前 1 行、段后 0.5 行 |
| 三级标题 | 左起空两字符，四号宋体，1.5 倍行距，段前 0.5 行、段后 0 行 |
| 四级以下 | 显目程度不超过上级，可用（1）、①、A.、a.；不进目录 |
| 正文 | 小四号宋体（英文数字 Times New Roman），1.5 倍行距 |
| 章节编号 | 理工类：1、1.1、1.1.1；人文社科类可用 一、（一）、1 |

## 前置部分版面

- **声明**：单独一页，“声明”二字三号黑体居中、二字间距两字符，段前 3 行段后 2 行；正文四号宋体，段首空两字符
- **中文摘要**：“摘要”同上格式；正文小四宋体 1.5 倍；正文后空一行接“关键词”（小四宋体加粗，三字中间空一字符），关键词 3-5 个、分号隔开、末尾无标点；摘要不少于 400 字符
- **英文摘要**：“ABSTRACT”三号 Times New Roman；正文小四 TNR 1.5 倍；“KEY WORDS”大写小四 TNR 加粗
- **目录**：“目录”三号黑体居中二字间距两字符；只列到三级标题；一级顶格、下级逐级缩一字符；标题与页码间用“……”连接，页码顶格右对齐；**用 Word 自动生成**（默认小四宋体、1.2 倍行距）

## 图、表、公式

- 编号一律**分章连续**：图 1-1、表 1-1、公式（1-1）
- **图**：图号+图题置于图**下方**居中，五号宋体加粗；图要有自明性；图中用字五号宋体（最小不小于七号）；分图用 (a)(b)(c)，分题注在各分图正下方
- **表**：表号+表题置于表**上方**居中，五号宋体加粗；**一律三线表**——顶线/底线 1.5 磅、中线 1 磅，可加辅助线；表格通栏与正文齐宽；表内容 1.5 倍行距；表中用字五号宋体；转页接排要重复表号加“（续）”并重复表头
- **公式**：另起一行居中，编号写在行末小括号（1-1），不加虚线；“式中：”顶格单独占行；量与单位按 SI/国标

## 引用与注释

- 参考文献**实引**：文中上角标 [1]、[2]…；同一文献多次引用始终用首次序号；同处多篇 [2，3] 或 [11-13]；多次引用的页码写在角标外（如 [1]32）
- 著录格式：GB/T 7714—2005；编号左起顶格，换行与前行文字对齐；中文文献用全角标点接排，英文文献用半角标点后空一格
- 数量：原则上 ≥10 篇，外文 ≥2 篇；作者 ≤3 全列，>3 列前三加“等”/“et al”
- **注释**（解释说明用，区别于引用）：上角标①②，脚注小五号宋体，与正文用 1/4 版宽细线隔开

## 文字细则

- 中文用国家公布的宋体简化字；英文和阿拉伯数字一律 Times New Roman
- 外文缩写首次出现注明含义：CPU（Central Processing Unit，中央处理器）
- 测量统计数据一律阿拉伯数字；非确数可用中文（“约一百五十人”）；“三力作用于一点”不写成“3力作用于1点”
- 单位符号全文统一（国际通用符号或中文名称，不混用）

## 篇幅与结构（毕业论文口径，学年论文参照）

- 字数不少于 8000 字
- 正文一般含绪论（前言）、正文、结论三部分；绪论要说明课题意义/目的/研究范围/技术要求、国内外概况与问题、研究思路、拟解决的主要问题
- 结论：归纳综合成果、与已有结果比较、指出价值、尚存问题与建议；概括、篇幅较短
- 题目 ≤35 汉字；摘要含选题意义/方法/结论，突出创新点

## 装订顺序（毕业论文，供参考）

封面 → 声明 → 任务书 → 中外文摘要 → 目录 → 正文 → 参考文献 → 致谢 → 附录 → 三份评议书
